# Pixel Art Maker Project
This project lets you draw the images using pixel boxes on the screen and then
it also lets you download the final drawn image (if you intend to do so).

## Getting Started
1. Clone the Project - `https://github.com/gauravsinghaec/FEND-pixel-art-maker.git`
2. Go into the directory where the project lives - `cd FEND-pixel-art-maker`
3. Open index.html in your favorite browser
4. Select colors and draw your desired art
5. Download the image if you like to reuse it.

## Folder Structure
Your project folder should look like this:
```sh
├── README.md -
├── designs.js # Javascript code for this project
├── html2canvas.min.js # html2canvas library to convert HTML element to canvas
├── index.html # Project Markup file
├── styles.css # Styles for this project.
└── udacity.png # Background image used in this project
```
